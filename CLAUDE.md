# Proyecto uigen
App de generación de UI con Next.js y Prisma.

## Comandos importantes
- npm run dev → inicia el servidor
- npm run setup → instala dependencias y base de datos

## Arquitectura
- Frontend: Next.js + Tailwind CSS
- Base de datos: SQLite con Prisma
- IA: Anthropic API
